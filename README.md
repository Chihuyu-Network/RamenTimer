# RamenTimer
Useful timer plugin for eating ramen.

## Commands

* `/ramentimer start <seconds> <title>`  
Start the timer.

* `/ramentimer end <title>`  
End the timer.

* `/ramentimer reload`  
Reload config.

## Permissions

* `ramentimer.ramentimer`  
Use /ramentimer

* `ramentimer.start`  
Use /ramentimer start

* `ramentimer.end`  
Use /ramentimer end

* `ramentimer.forceend`  
End other's timer forcely.

* `ramentimer.reload`  
Use /ramentimer reload
